\doxysubsection{Detailed Description}
A concrete command for customizing an order with specific customization details. 

\doxylink{classCustomizeOrder}{Customize\+Order} is a specific command that represents the action of customizing an order at a restaurant. It is associated with a particular \doxylink{classWaiter}{Waiter} and allows customization of a given \doxylink{classRestaurantTable}{Restaurant\+Table}\textquotesingle{}s order.
