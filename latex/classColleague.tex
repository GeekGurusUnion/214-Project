\doxysubsection{Detailed Description}
Abstract base class for colleagues in the \doxylink{classMediator}{Mediator} pattern. 

\doxylink{classColleague}{Colleague} represents a participant in the \doxylink{classMediator}{Mediator} pattern, which collaborates with other colleagues through a mediator to achieve a common goal.
