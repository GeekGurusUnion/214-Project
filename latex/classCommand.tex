\doxysubsection{Detailed Description}
Abstract base class for commands in the \doxylink{classCommand}{Command} pattern. 

\doxylink{classCommand}{Command} represents a command to be executed, typically associated with a specific action, and it decouples the sender (\doxylink{classWaiter}{Waiter}) from the receiver (\doxylink{classRestaurantTable}{Restaurant\+Table} and \doxylink{classMenuItem}{Menu\+Item}).
