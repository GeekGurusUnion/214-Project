\doxysubsection{Detailed Description}
A concrete \doxylink{classCommand}{Command} that represents a tip order given by a customer to a waiter. 

This class inherits from the \doxylink{classCommand}{Command} abstract class and implements the execute method. The execute method takes a \doxylink{classRestaurantTable}{Restaurant\+Table} and a double representing the tip amount as parameters. When executed, it calls the add\+Tip method of the \doxylink{classWaiter}{Waiter} passed in the constructor with the given tip amount.
