\doxysubsection{Detailed Description}
An interface representing an iterator for a collection of elements. 

\doxylink{classIterator}{Iterator} provides a common interface for iterating over elements in a collection without exposing the underlying structure of the collection. Concrete iterator classes implement these methods for specific collections.
